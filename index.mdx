## What is Versive?

Versive is a comprehensive ecosystem designed to facilitate interview processes. It is a mono-repository containing several key components:

<CardGroup cols={3}>
  <Card title="Interviewer" icon="browser">
    A Next.js application for the frontend interface.
  </Card>
  <Card title="Versive API" icon="server">
    A FastAPI backend service (Python).
  </Card>
  <Card title="Versive DB" icon="database">
    Supabase database configuration and migrations.
  </Card>
</CardGroup>
